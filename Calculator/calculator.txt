On Day 11, I built a Calculator App using React.
The design was inspired by my watch’s calculator, and I used a YouTube tutorial as a starting point.

Initially, I wrote the entire application in a single file, but later refactored it into multiple components to improve structure, readability, and reusability.

⚙️ Features
➕ Perform basic arithmetic operations (+, −, ×, ÷)
🔢 Display live input and results
⌫ Delete last character
🧹 Clear entire input
⚡ Instant calculation on "="
🧩 Component Structure
1. App.jsx
Main component managing state and logic
Stores user input using useState
Handles all calculator operations

Example:

🔁 Core Functions (App Logic)

From App.jsx:

🔹 handleClick(value)
Adds numbers/operators to input
setInput((prev) => prev + value);
🔹 handleDelete()
Removes last character
setInput((prev) => prev.slice(0, -1));
🔹 handleClear()
Clears entire input
setInput("");
🔹 handleEquals()
Calculates result using eval()
setInput(eval(input).toString());
Uses try-catch to handle errors
⚠️ Important Learning
🚫 Why eval() is risky
Can execute malicious code
Causes security vulnerabilities
Not recommended for production apps

👉 Learned that better alternatives exist (like expression parsers)
